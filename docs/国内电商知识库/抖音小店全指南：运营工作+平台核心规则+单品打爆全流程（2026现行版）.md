## 一、抖音店铺运营全工作内容（日常 \+ 月度全模块）

### （一）日常基础运营（每日必做）

1. **数据复盘（罗盘后台）**
查看前一日 GMV、访客、点击率、转化率、商品卡流量、搜索流量；核对体验分、品退率、差评、负反馈，数据下滑定位：大盘下行 / 单品权重下跌 / 差评拉低流量 / 活动下线。

2. **商品管理**
优化标题（埋罗盘热搜词）、主图、sku、详情；滞销品调价 / 下架，新品补上架；管控价格体系，杜绝站内比价违规。

3. **售后 \& 飞鸽客服**
飞鸽 5 分钟响应率达标，处理退换货、投诉、差评维护；7 天无理由履约，异常订单拦截止损，压低品退率（品退＞5% 直接限制商城流量）。

4. **店铺分值巡检**
体验分、商品质量分、物流分、服务分四项，低于 60 分无法报名商城活动、无法入猜你喜欢推荐。

### （二）货品运营（选品 \+ 货盘规划）

1. 依托**电商罗盘 \- 行业榜单、飙升榜、搜索词榜**选品，分 3 类布局：

- 引流款：低毛利、性价比，用来破 0、拉店铺权重

- 利润款：主力盈利，打爆核心单品

- 形象款：高客单，拉高店铺客单价

2. 搭建 sku 阶梯定价，设置搭配购、满减，提升连带率。

### （三）流量运营（内容 \+ 货架 \+ 付费 \+ 达人）

1. **货架流量（商品卡 / 商城 / 搜索）**：优化商品卡点击率、关键词排名，报名商城日常活动、品类秒杀、百亿补贴；开通商品卡免佣拿自然曝光。

2. **内容流量（短视频 \+ 直播）**：自运营账号日更 3\~5 条产品短视频；日播 / 间歇直播承接即时流量，优化直播间转化率。

3. **付费投放**：千川 / 小店随心推，分测款投放、放量投放，把控 ROI；商品卡付费撬动搜索自然流量。

4. **达人分销**：入驻精选联盟，设置佣金，对接纯佣 / 坑位费达人，中小达人铺量、头部达人冲爆发。

### （四）活动 \& 营销（月度重点）

1. 平台大促：618、双 11、平台品类日、超值购、秒杀报名；店铺自建：满减、优惠券、拼团、会员权益。

2. 会员运营：店铺会员储值、复购券，提升老客复购，降低获客成本。

### （五）供应链 \& 财务

对接仓库把控发货时效（明日达规则）、库存预警；核算投产、毛利、推广成本，调整备货量。

二、单品完整打爆 SOP 全流程（7\~15 天从新品→爆款，全域打法：货架 \+ 内容 \+ 付费 \+ 达人）

### 阶段 1：测款筹备期（1\~3 天，基础基建）

1. **产品上架优化**
标题埋 5\~8 个罗盘热搜关键词，主图 5 张分层：痛点→产品→细节→场景→售后保障；详情突出差异化卖点；设置运费险、sku 价格阶梯、优惠券。

2. 开通精选联盟，设置合理佣金（引流款 15%\\30%，利润款 10%\\20%）；配置运费模板、店铺活动。

3. 筹备 3\~5 条差异化短视频脚本：痛点款、测评款、场景实拍款（用于 A/B 测素材）。

### 阶段 2：冷启破 0 期（3\~5 天，小预算测数据）

1. **免费打底**：每日发布 3 条短视频，挂载商品；商品卡优化，蹭搜索自然流量；私域老客破 0，10\~30 单基础销量 \+ 5 条带图好评（无违规 S 单）。

2. **小额付费测素材**：单条短视频随心推 30\~80 元，目标成交，单素材总预算≤500 元；
**筛选标准**：3 秒完播＞60%、商品点击＞8%、加购＞5%= 优质素材，留存；数据极差直接淘汰素材。

3. 小达人铺量：10\\30 位千粉纯佣达人带货，冲 50\\200 单基础销量，拉高商品初始权重。

### 阶段 3：权重爬坡期（5\~9 天，付费放量 \+ 货架提权）

1. **优质素材千川放量**：优选 2 条最优素材开千川 GMV Max 计划，小额多计划投放，控制投产≥1\.8；付费撬动商品卡、搜索自然流量。

2. **货架运营**：报名超值购、品类秒杀低价活动，快速冲销量，提升商品搜索排名；持续维护好评，差评当天处理整改。

3. **数据优化**：罗盘盯搜索词，新增高转化关键词补充标题；高退货溯源：产品 / 包装 / 物流，同步优化供应链。

> 达标标志：日销 300\~800 单，自然流量占比＞40%，商品进入类目小榜。
