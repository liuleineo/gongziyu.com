# 电商行业主流分类维度与典型代表

电商行业主流按**交易主体、流量模式、商品属性、交易范围、供应链模式**五大维度划分，下面是结构化总览与典型代表。

---

### 一、按交易主体（最经典）

- **B2C（企业对个人）**：平台型（天猫、京东）、自营型（京东自营、网易严选）、DTC 品牌（完美日记、Allbirds）。

- **B2B（企业对企业）**：批发 / 采购平台（1688、慧聪网）、产业带电商（找钢网、西域）、供应链服务（联塑商城）。

- **C2C（个人对个人）**：二手 / 闲置（闲鱼、转转）、个人小店（淘宝个人店）、手作 / 文创（微店）。

- **C2B（个人对企业）**：定制团购（必要商城）、反向定价（拼多多拼单）、设计师平台（红布林）。

---

### 二、按流量 / 运营模式（主流新业态）

- **货架电商（搜索电商）**：用户主动搜索，类目化陈列（淘宝、京东、拼多多）。

- **内容 / 兴趣电商**：短视频 / 图文种草，算法推荐（抖音、小红书、视频号）。

- **直播电商**：主播实时讲解 \+ 互动转化（抖音直播、快手直播、淘宝直播）。

- **社交电商**：社群裂变 / 熟人推荐（拼多多、云集、微商）。

- **即时零售（本地电商）**：线上下单、1 小时达（美团闪购、京东到家、饿了么）。

- **全渠道电商**：线上线下打通（银泰百货、苏宁易购、盒马）。

- **订阅电商**：周期购 / 会员制（每日优鲜、叮咚买菜、花点时间）。

---

### 三、按商品属性（垂直 / 综合）

- **综合电商**：全品类（天猫、京东、拼多多）。

- **垂直电商**：聚焦单一品类

    - 生鲜（叮咚买菜、朴朴）

    - 美妆（唯品会、聚美优品）

    - 服饰（蘑菇街、得物）

    - 家电（京东家电、苏宁易购）

    - 奢侈品（寺库、红布林）

    - 医药（1 药网、京东健康）

---

### 四、按交易范围（国内 / 跨境）

- **国内电商**：面向大陆市场（淘宝、京东、抖音）。

- **跨境电商**

    - **出口跨境**：B2C（SHEIN、Temu、Anker）、B2B（阿里巴巴国际站）。

    - **进口跨境**：保税仓 / 直邮（天猫国际、京东国际、考拉海购）。

---

### 五、按供应链 / 货源模式

- **有货源电商**：自有工厂 / 仓库，自主发货（品牌旗舰店、京东自营）。

- **无货源电商**：一件代发（1688 代发、拼多多无货源店群）。

- **一件代发（分销）**：供应商直发，零库存（微商、抖音小店分销）。

- **仓播电商**：仓库直播，现货直发（家具、家电、服饰仓播）。

---

### 六、其他细分类型

- **社区团购**：团长社群预售 \+ 集中配送（兴盛优选、多多买菜）。

- **二手电商**：闲置交易（闲鱼、转转、爱回收）。

- **拼团电商**：低价拼单（拼多多、美团优选）。

- **返利 / 导购电商**：佣金分成（什么值得买、返利网）。

---

### 总结

电商分类可概括为：**主体（B2C/B2B/C2C）\+ 流量（货架 / 内容 / 直播）\+ 商品（综合 / 垂直）\+ 范围（国内 / 跨境）\+ 供应链（有 / 无货源）**。
