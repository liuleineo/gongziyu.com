你想了解 Python 中的数组（对应 Python 里的**列表 `list`**）排序方法，我会和 JavaScript 的 `sort()` 做类比讲解，让你更容易理解和区分，同时覆盖实际开发中的常用场景。

Python 中列表排序主要有两种核心方式，分别对应不同的需求：
1.  `list.sort()` 方法（原地排序）
2.  `sorted()` 内置函数（返回新列表，不修改原列表）

### 一、 方式1：`list.sort()` - 原地排序
这是列表对象的内置方法，和 JavaScript 的 `arr.sort()` 类似，**会直接修改原始列表，没有返回值（或返回 `None`）**。

#### 1. 基础用法（无额外参数）
```python
# 数字列表
num_list = [10, 2, 30, 4]
# 字符串列表
str_list = ["b", "a", "d", "c"]

# 无参数排序
num_list.sort()
str_list.sort()

print(num_list)  # [2, 4, 10, 30]（Python 直接按数值大小排序，无需手动写比较函数，比 JS 友好）
print(str_list)  # ["a", "b", "c", "d"]（按字符串字母/Unicode 顺序排序）
```

**关键说明**：
- 无参数时，`sort()` 对**数字列表**按「数值大小升序」排序，对**字符串列表**按「字母/Unicode 顺序升序」排序，无需像 JS 那样担心多位数排序异常。
- 核心特点：**原地修改**，原始列表会被改变，方法本身返回 `None`（不要写 `new_list = num_list.sort()`，否则 `new_list` 会是 `None`）。
- 支持的元素类型：列表中的元素必须是「可比较的同一类类型」（比如不能同时包含数字和字符串，否则会报错）。

#### 2. 进阶用法（指定参数定制排序规则）
`list.sort()` 支持两个常用可选参数，来满足复杂排序需求：
- `reverse`：布尔值（`True`/`False`），是否降序排序，默认 `False`（升序）。
- `key`：接收一个**函数**作为参数，用于指定「排序的依据」，这个函数会作用于列表中的每一个元素，返回值作为该元素的「排序关键字」。

##### 场景1：数字列表降序排序
```python
num_list = [10, 2, 30, 4]
# reverse=True 开启降序
num_list.sort(reverse=True)
print(num_list)  # [30, 10, 4, 2]
```

##### 场景2：对象列表（字典列表）按指定属性排序
这对应你之前 JS 中按 `totalFee` 排序的场景，Python 中用 `key` 参数实现，更简洁且无需手动做类型转换容错（可结合 `lambda` 匿名函数快速实现）。

```python
# 字典列表（对应 JS 中的对象数组）
order_list = [
    {"name": "订单1", "totalFee": "299.9"},
    {"name": "订单2", "totalFee": "150"},
    {"name": "订单3", "totalFee": "abc"},  # 无效值
    {"name": "订单4", "totalFee": "500"}
]

# 按 totalFee 数值升序排序（key 指定排序依据，lambda 函数处理每个元素）
order_list.sort(
    key=lambda x: float(x["totalFee"]) if x["totalFee"].replace(".", "", 1).isdigit() else 0,
    reverse=False
)

print("按 totalFee 升序：", order_list)
```

**说明**：
- `lambda x: ...` 是匿名函数，`x` 代表列表中的每一个字典元素，返回值作为该元素的排序关键字。
- `x["totalFee"].replace(".", "", 1).isdigit()` 是容错逻辑，判断 `totalFee` 是否是有效数字（支持小数），无效则返回 `0`，避免 `float()` 报错。
- 对比 JS：Python 无需写复杂的比较函数，`key` 参数直接指定排序依据，更直观简洁。

##### 场景3：按字符串长度排序
```python
str_list = ["apple", "banana", "pear", "orange"]
# 按字符串长度升序排序，key 为 len 函数（获取字符串长度）
str_list.sort(key=len)
print(str_list)  # ["pear", "apple", "banana", "orange"]
```

##### 场景4：多条件排序（优先级排序）
比如：先按年龄升序，年龄相同则按薪资降序，通过 `key` 返回元组实现（元组会按元素顺序依次比较）。

```python
user_list = [
    {"name": "张三", "age": 25, "salary": 15000},
    {"name": "李四", "age": 30, "salary": 20000},
    {"name": "王五", "age": 25, "salary": 18000},
]

# 先按 age 升序，再按 salary 降序（salary 取负数实现降序，因为整体是升序排序）
user_list.sort(
    key=lambda x: (x["age"], -x["salary"])
)

print("多条件排序结果：", user_list)
# 结果：张三（25/15000）→ 王五（25/18000）→ 李四（30/20000）
```

### 二、 方式2：`sorted()` - 返回新列表排序
这是 Python 的内置函数，**不会修改原始列表，而是返回一个排序后的新列表**，这是和 `list.sort()` 最核心的区别。

#### 基本用法和进阶用法
`sorted()` 的使用语法和 `list.sort()` 高度一致，同样支持 `reverse` 和 `key` 参数，唯一区别是它可以作用于「所有可迭代对象」（不仅限于列表，还包括元组、字符串等）。

```python
num_list = [10, 2, 30, 4]
# 升序排序，返回新列表，原始列表不变
sorted_num_asc = sorted(num_list)
# 降序排序
sorted_num_desc = sorted(num_list, reverse=True)

print("原始列表：", num_list)  # [10, 2, 30, 4]（未被修改）
print("升序新列表：", sorted_num_asc)  # [2, 4, 10, 30]
print("降序新列表：", sorted_num_desc)  # [30, 10, 4, 2]

# 对字典列表使用 key 参数（和 list.sort() 一致）
order_list = [
    {"name": "订单1", "totalFee": "299.9"},
    {"name": "订单2", "totalFee": "150"},
]
sorted_order = sorted(
    order_list,
    key=lambda x: float(x["totalFee"]) if x["totalFee"].replace(".", "", 1).isdigit() else 0
)
```

### 三、 Python 排序 vs JavaScript 排序（核心区别）
1.  **默认排序逻辑**：
    - JS：无参数 `sort()` 按「字符串 Unicode」排序，数字排序会出错。
    - Python：无参数 `sort()`/`sorted()` 按「数值大小（数字）」或「字母顺序（字符串）」排序，更符合直觉。
2.  **排序方式**：
    - JS：只有 `arr.sort()` 原地排序，想要保留原数组需要手动拷贝。
    - Python：提供 `list.sort()`（原地）和 `sorted()`（返回新列表），按需选择，更灵活。
3.  **复杂排序实现**：
    - JS：通过传入「比较函数」（`a, b`），靠返回值控制排序。
    - Python：通过 `key` 参数指定「排序依据」，更简洁高效，无需手动处理比较逻辑。

### 四、 补充场景：中文排序
Python 默认也不支持中文拼音排序，需要借助第三方库 `pypinyin` 实现：
1.  先安装库：`pip install pypinyin`
2.  实现排序：
```python
from pypinyin import lazy_pinyin

chinese_list = ["张三", "李四", "王五", "赵六"]
# 按中文拼音升序排序
chinese_list.sort(key=lambda x: "".join(lazy_pinyin(x)))

print(chinese_list)  # ["李四", "王五", "张三", "赵六"]
```

### 总结
1.  Python 列表排序有两种核心方式：`list.sort()`（原地修改，无返回值）和 `sorted()`（返回新列表，不修改原数据）。
2.  两个方式都支持 `reverse`（控制升降序）和 `key`（指定排序依据）参数，`key` 结合 `lambda` 函数可解决对象列表等复杂排序场景。
3.  Python 默认排序逻辑更符合直觉，复杂排序实现比 JS 更简洁，按需选择原地排序或新列表排序即可。
