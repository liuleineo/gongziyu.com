什么是NumPy？

NumPy（Numerical Python）是Python科学计算的核心库，提供了高效的多维数组对象和数学函数，是数据科学、机器学习和科学计算的基础工具。

# 数据标准化

实战项目：房价预测数据分析

# 导入必要的库
import numpy as np
import matplotlib.pyplot as plt

# 生成模拟数据
np.random.seed(42)
house_size = np.random.randint(50, 200, 100)
house_price = 30 * house_size + np.random.randn(100) * 200

# 数据预处理
X = house_size.reshape((-1, 1))
y = house_price

# 添加偏置项
X_b = np.c_[np.ones((100, 1)), X]

# 使用最小二乘法求解线性回归
theta_best = np.linalg.inv(X_b.T.dot(X_b)).dot(X_b.T).dot(y)

print("回归系数:", theta_best)

# 预测
X_new = np.array([[50], [200]])
X_new_b = np.c_[np.ones((2, 1)), X_new]
y_predict = X_new_b.dot(theta_best)

print("预测结果:", y_predict)

# 可视化结果
plt.scatter(X, y)
plt.plot(X_new, y_predict, "r-")
plt.xlabel("房屋面积(平方米)")
plt.ylabel("房屋价格(万元)")
plt.show()
