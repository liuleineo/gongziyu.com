# 一、核心工作是干啥？（一句话）

**设计、搭建、调试、维护自动化产线/设备，让机器代替人干活。**

日常主要干 5 件事：

1. **设备/产线方案设计**

机械手、传送带、工装夹具、检测机构、上下料机构

1. **机械结构设计 + 出图**

3D建模、装配、零件图、BOM、加工图纸

1. **电气/控制配合**

跟电气工程师对接：传感器、气缸、电机、PLC点位

1. **现场调试、安装指导**

去工厂装设备、调精度、解决卡料、抖动、定位不准

1. **售后维护、改善优化**

设备坏了修、效率低了改、故障率高了优化

# 二、必须会的硬技能（企业最看重）

## 1. 机械设计能力

- 看懂并设计**机构**：连杆、凸轮、导轨丝杆、皮带线、齿轮箱

- 懂**公差配合、材料选型、热处理、表面处理**

- 会**受力分析、动平衡、刚性计算**

## 2. 软件技能（必学 4 件套）

- **SolidWorks**（90% 企业用）

- **UG / Creo**（汽车/模具大厂）

- **AutoCAD**（出2D工程图）

- **CAE 仿真（可选）**：ANSYS/ABAQUS 做强度、振动分析

## 3. 自动化核心部件（必须认识+会选）

- **气缸、电缸、滑台**

- **伺服电机、步进电机、减速机**

- **导轨、丝杆、联轴器、轴承**

- **传感器**：光电、接近、光纤、视觉相机

- **气动系统、液压系统**

## 4. 电气基础（非常重要！）

不需要你写程序，但必须懂：

- PLC 工作逻辑

- 传感器 I/O 信号

- 电机驱动、伺服参数

- 气动回路、电磁阀

不然你设计的机构，电气根本没法控制。

## 5. 加工与工艺知识

- 车铣刨磨、线割、CNC、钣金、压铸

- 知道什么零件怎么做、成本多少、能不能加工

- 避免设计出“图纸很美，造不出来”的东西

## 6. 现场调试能力（最值钱）

- 设备装配

- 精度调整

- 故障排查（卡料、抖动、错位、异响）

- 节拍优化（提高效率）

**能现场解决问题 = 工资直接上一个档次**

# 三、软实力（决定你能不能升主管）

- 项目推进能力

- 跨部门沟通（电气、采购、生产、客户）

- 问题分析与根因查找（5Why、鱼骨图）

- 方案汇报、PPT 讲解

# 四、职业发展路线（真实）

1. 助理工程师 → 画图、拆图、打杂

2. 机械工程师 → 独立设计设备、出方案

3. 高级工程师 → 负责整线方案、复杂机构

4. 技术主管/经理 → 管项目、管人

5. 技术总监 → 定技术路线、新产品规划

---

# 五、一句话总结

**机械自动化工程师 = 机械设计 + 自动化部件 + 电气逻辑 + 现场调试**

核心价值：

**让设备稳定、高效、不出故障地自动运行。**
